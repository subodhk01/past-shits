Header Files

Header Files contains pre-defined functions which can be further used by users. Till now we have discussed 2 header files :   1) stdio.h     2) conio.h

stdio.h

This header file includes functions like printf(), scanf(), etc.

  • printf()

    • This is a very basic and most important function in C. It is used to print anything on the output screen.

    Usage - printf("Message");
    but for printing a variable we cannot directly write the variable's name in the inverted commas but we have to do something different.
    ex- for a int variable a, we have to do

    printf("Sum = %d",a);

    Here %d is used to save that place for the value of any integer and at end we have to specify the variable.

    For character, we use %c
    For string, we use %s
    For float, we use %f

    But if we want to add multiple as well as different types of variables in one printf() statement, we do so

    printf("Values are %d %f %s",a,b,c);

  • scanf()

    • scanf() is another function in stdio.h header file. It is another very important function in C. It is used to take input(s) from the user.

    int a;
    scanf("%d",&a);

    scanf() is quite similar to printf() but scanf() requires only %d %f %s %c inside inverted commas and here with the variable name we have to use a "&" sign. In C this sign is used to direct address. It is neccesary to use "&" sign with scanf() function.
    We can also input several datas at one time also.

    int a,b;
    scanf("%d%d",&a,&b);

    NOTE : Be proper with the syntax of scanf(), if you even added an extra space, your progran will not execute properly.

conio.h

This header file includes functions like clrscr(), getch() etc.

  • clrscr()

    • This function is used to clear the output screen. This is an optional function rather this whole header file is optional but is used for special purposes.This function do not have any parameters, we just have to write clrscr().

  • getch()

    • This funtion is basically used to hold the output screen. It is also an optional function but I recommend using it everytime as not using it may lead to confusion coz you will not notice anything. I suggest to compile any program without using getch() and you will automatically understand everything.
