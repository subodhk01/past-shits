Arrays

Arrays is a type of variable which is used to store multiple values of same type. Instead of declaring num1 = something ; num2 = something ; num3 = something ; , we can make an array and put all our values in it

  • Syntax:
  • datatype name[value];
    •

    Arrays are declared similarly as any variable but in arrarys we have to declare its size too. The amount of data which our array can store depends on the value.

  • int num[5];
  • num[0] = 12;
  • num[1] = 345;
  • num[2] = 700;
  • num[3] = 34;
  • num[4] = 33;
    •

    Here num is our array and the number of integers which it can store is 5. Remember that arrays are zero indexed i.e they start from (0) and end at (n-1) where n is the size of the array.

    NOTE : Strings are also a type of array of characters but if we want to make a array of string, we can do so.

    char  name[20][5];

    Here [20] makes the array of character which forms a string and then [5] is used to form the array of that string.
    We can also use scanf() to insert value as so :
    scanf("%d",&num[3]);
