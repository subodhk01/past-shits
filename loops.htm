1. While Loop

While loop is a basic type of loop. It is used to execute the statements till the condition remains matched.

Syntax:

  • while(condition)
  • {
  • statement;
  • ++/--;
  • }
    •
    Here ++/-- means use of increment or decrement oprators.

    NOTE : Always check your loops before executing it. Avoid Infinite loops.

  • #include<stdio.h>
  • #include<conio.h>
  • void main()
  • {
  • int n;
  • clrscr();
  • n = 1;
  • while( n<=5 )
  • {
  • printf("%d \n",n);
  • n++;
  • }
  • getch();
  • }

    2. Do while Loop

    Do while is similar to while loop but in do while loop the program gets executed at least one time and then it depends on the condition.

  • Syntax:
  • do
  • {
  • statement;
  • ++/--;
  • }while(condition)

  • #include<stdio.h>
  • #include<conio.h>
  • void main()
  • {
  • clrscr();
  • int n = 1;
  • do
  • {
  • printf("%d \n",n);
  • n++;
  • }while(n<=10);
  • getch();
  • }
  •

    3. for loop

    for loop is the most famous as well as the mostly used type of loop. for loop also does the same thing but there is a difference in the syntax.

  • Syntax:
  • for( starting variable ; condition ; ++/-- )
  • {
  • statements;
  • }
    •

  • #include<stdio.h>
  • #include<conio.h>
  • void main()
  • {
  • clrscr();
  • int n = 1;
  • for( n ; n<=10 ; n++ )
  • {
  • printf("%d \n",n);
  • }
  • getch();
  • }
