1. Simple if

It is used to execute the statement according to the given specific contition.

Syntax:                      if(condition)

2. if else

It is used to execute the statement where the condition is either true or false.

Syntax:                    if(condition)
                                         statement;
                 else
                                         statement;
Here is the next version of the program which we have previously done.

  • else
  • statement;
  • }

    5. Switch

    It is a bit different type of conditional statement. Switch also does the same work which other conditionals do but is works to check conditions of only one varialbe.

  • Syntax:
  • switch(variable)
  • {
  • case value:
  • statement;
  • break;
  • case value:
  • statement;
  • break;
  • default:
  • statement;
  • }
    •
    Here you can see that after every statement we have used break statement. break is type of unconditional statement, we will learn about it later. Here default statement has its own purpose, when neither condition gets matched the default statement gets executed. You will understand it much better while doing it practically.
