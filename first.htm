Our First Program

As C program need to be compiled, I would suggest you to use Turbo C++ to compile the programs. You can download it from here.
Before procedding towards writing our program, here are some shortcut keys for Turbo C++ which will help you.

F9 To compile the program.
Ctrl + F9 To compile and run the program.
Alt + Enter To exit full screen mode.

So, our program will be adding two numbers.
#include<stdio.h>
#include<conio.h>

               void main()
{
               clrscr();
               int a, b, c ;
               a = 10 ;
               b = 20 ;
               c = a + b ;
               printf( "Sum = %d", c);
               getch();
}

Here is a brief description about our program :

#include<stdio.h> #include<conio.h> #include is used to add header files to our program. Here stdio.h and conio.h are the header files.
void main() {} main() is a compulsary function in C, we will learn about functions later.
clrscr(); This function is used to clear the output screen. We will discuss about it later.
int a,b,c; Here we have declared 3 variables(a,b and c) and declared their type as int i.e. Integer
a = 10; b = 20; Here we have assigned the values of a and b as 10 and 20.
c = a + b Here with the use of operator we have assigned the value of c to the sum of a and b.
printf("Sum = %d",c); Here we are printing the value of c. We will discuss about printf() function later.
getch(); This function is used to hold the screen.
