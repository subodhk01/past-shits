What is C?

C is basically a very popular programming language. It was designed by a man named Dennis Ritchie. C is very simple, easy to use and reliable. The speed of execution of programs in C is very fast. This is all for a begginer to learn about C. Now let us learn about the language Basics of C and compare it with English Language.

Language Basics

Not only in C but in almost every language, may be it is a programming language or a human communication language, every language is similar. If we say in the reference of english each language has its own alphabets, words, sentences, paragraphs and stories too.
