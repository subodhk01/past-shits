Before discussing about the types of functions, I want you to know about two terms.

1. Parameters

Parameters are simply the text inside the function's brackets.

NOTE : It is not neccesary that every function has parameters. Ex- clrscr() , getch() .

2. Return

return statement declares a value that the particular fuction will return. You will get it more clearly when you will do it practically.

Types of functions

Basically there are four types of functions in C :
1. Function with no parameter and no return value.

This type of function is similar to the function which we did in the previous chapter.

  • Syntax :
  • void   name();
  • {
  • code;
  • }

  • #include<stdio.h>
  • #include<conio.h>
  • void add()                //This is our function
  • {
  • int a,b,c;
  • printf("Enter Two Numbers :");
  • scanf("%d%d",&a,&b);
  • c = a + b ;
  • printf("Sum = %d",&c):
  • }
  • void main()
  • {
  • clrscr();
  • add();
  • getch();
  • }
  •

    2. Function with parameter but no return value.

  • Syntax :
  • void   name(datatype name, datatype name,....);
  • {
  • code;
  • }

  • #include<stdio.h>
  • #include<conio.h>
  • void add(int a, int b)
  • {
  • int c;
  • c = a + b ;
  • printf("Sum = %d",c);
  • }
  • void main()
  • {
  • clrscr();
  • add( 20 , 40 );
  • getch();
  • }
  •

    Since this function is with parameters we have to provide certain parameters and here those parameters are int a and int b. Now by the parameters the function requires two int values and we have provided those values in the main() function i.e. 20 and 40.
    We can also use variales as parameters.

    3. Function with no parameter but a return value.

  • Syntax :
  • datatype   name();
  • {
  • code;
  • return something ;
  • }

    • Notice that here void is not used because this function is going to return some value. ( void means - " nothing " )
  • #include<stdio.h>
  • #include<conio.h>
  • int add()
  • {
  • int a , b , c ;
  • printf("Enter Two Numbers :");
  • scanf("%d%d",&a,&b);
  • c = a + b ;
  • return c;
  • }
  • void main()
  • {
  • clrscr();
  • int x = add();
  • printf("\n Answer = %d",x);
  • getch();
  • }
  •

    Here in this program, the add() function returns a value c and then inside the main() function that value of c gets stored in x.

    4. Function with both parameter and return value.

  • Syntax :
  • datatype   name(datatype name, datatype name,....);
  • {
  • code;
  • return something ;
  • }

  • #include<stdio.h>
  • #include<conio.h>
  • int add(int a, int b)
  • {
  • int c;
  • c = a + b ;
  • return c;
  • }
  • void main()
  • {
  • int x,y,z;
  • clrscr();
  • printf("Enter Two Numbers :");
  • scanf("%d%d",&x,&y);
  • z = add(x,y);
  • printf("\n Answer = %d",z);
  • getch();
  • }
