Variables

Variable is a user defined name which can store values. This variable is same as we study in algebra in Mathematics.

ex: x,y,p,q all are variables

Declaring a variable

Syntax:  datatype   name;

At place of datatype we have to define the type of value which our variable will store,

for numbers - int
for character - char
for string - char[]
for decimal values - float

ex:   int x;               char ram[10];               float y;

Rules for declaring variables

  • It should start with a character, can be followed by digits.
  • It doesn't contain any spaces.
  • It is case sensitive.

Keywords

Keywords are the reserved words which cannot be used as variables.

ex:    int ,   char ,   if ,   switch   etc.

Operators

In simple language, opertors are something witch perform operations.

ex:  a + b = c

Here a,b,c are operands and + and = are operators.

Types of Operators

  • Arithematic Oprators

    • ex:   +   -   *   /   %      are arithematic operators

  • Increment or Decrement Operators

    • ++   and  --  are increment and decrement operators, respectively.

    ex:     1++  means  2

    and

    ex:     5--  means  4

  • Assignment Operator

    •   =   is the only assignment operator

    ex:  a  =  6 ;

  • Relational operators

    • <      (Less than)
    >       (Greater than)
    <=       (Less than Equal to)
    >=       (Greater than Equal to)
    ==       (Equals to)
    !=      (Not Equal to)

    ex:   6  ==  6                       7  !=  2                  2  <  7

  • Logical Operators

    • &&   means   and
    ||   means   or
