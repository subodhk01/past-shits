1. goto

goto is used to jump to any particular statement.

  • Syntax:
  • x:
  • statement;
  • goto x;
    •

    Here x is a label or we can say that it is a checkpoint. As soon as the goto statement executes the execution shifts to the label.

    NOTE : Never use goto in a normal way, if you do so the program will convert into infinite loop and will never end.

    NOTE : Only use colon(:) after label.

    Now here is a program to print numbers from 1 to 5.
  • #include<stdio.h>
  • #include<conio.h>
  • void main()
  • {
  • int n;
  • clrscr();
  • n=1;
  • x:
  • if( n<= 5)
  • {
  • printf("%d\n",n);
  • n++;
  • goto x;
  • }
  • getch();
  • }
  •

    NOTE : \n inside the printf() is used to shift the output to the next line. Use it once and you will get to know everything.

    2. return

    return is used to return a value to the function. We will discuss about it later when we will study about functions.

    3. continue

    As the name specifies, continue is used to continue any statement or conditional. It basically does nothing but instructs the program to leave the statement to go ahead.

  • #include<stdio.h>
  • #include<conio.h>
  • void main()
  • {
  • clrscr();
  • int n;
  • n=1;
  • while( n<=10 )
  • {
  • if( n==5 )
  • continue;
  • else
  • printf("%d\n",n);
  • n++;
  • }
  • getch();
  • }
    •

    Don't get confused by the while statement, its a type of loop, we will learn about it in the next chapter. Here it was just used to explain the continue statement.

    4. break

    break is similar to continue but the difference is that when break gets executed it completely terminates the loop or conditional. If you remember we used break in switch. We used it just as to terminate the switch as soon as our condition gets matched.
